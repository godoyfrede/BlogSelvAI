# A Revolução do App Router

O Next.js mudou o paradigma. A divisão entre Server Components e Client Components força os desenvolvedores a pensarem sobre a fronteira do dado e a renderização.

## Server Actions

Com a eliminação da necessidade de rotas de API `.ts` tradicionais para mutações, as Server Actions aceleram o desenvolvimento e reduzem o boiler-plate de requisições. No entanto, é crucial ainda focar na validação com ferramentas como Zod.
